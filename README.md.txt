# Healthcare Appointment Intelligence Platform

## Overview
A healthcare analytics project focused on understanding patient appointment attendance patterns and identifying factors contributing to no-shows.

## Technologies Used
- MySQL
- SQL
- Python
- Pandas
- Power BI

## Dataset
Medical Appointment No-Show Dataset (110,527 records)

## Key Insights
- Overall No-Show Rate: 20.19%
- Average Age: 37.09
- Young adults (19–35) have the highest no-show rate.
- Scholarship recipients show higher no-show rates.
- Female patients account for most appointments.
