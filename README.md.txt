# Healthcare Appointment Intelligence Platform

## Overview

A full-stack Data Analytics and Machine Learning project that predicts whether a patient is likely to miss a healthcare appointment.

The project combines:

- Data Cleaning
- SQL Analysis
- Power BI Dashboarding
- Machine Learning Prediction
- Flask Web Application
- MySQL Database Integration

---

## Features

- Predict patient no-show probability
- Store prediction history in MySQL
- Interactive Power BI dashboard
- Machine Learning model using Random Forest
- Web interface built with Flask
- Prediction logging and history tracking

---

## Tech Stack

- Python
- Pandas
- Scikit-Learn
- Flask
- MySQL
- Power BI
- Git & GitHub

---
